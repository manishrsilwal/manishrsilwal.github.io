Nepali Placeholder Kit

A Figma plugin that fills your designs with real Nepali names, addresses, and dates instead of the usual American defaults.

Role Designer & Developer
Company Personal Project
Timeline Personal project
Type Figma Plugin Development

The Problem

Every design tool ships with placeholder data that’s clearly American. John Smith, 123 Main Street, (555) 867-5309. None of that makes sense if you’re designing for Nepal. You end up manually typing Nepali names and addresses into every single text field, which is slow and breaks your flow.

What I Did

I built a Figma plugin that replaces all of that with a single click. Real Nepali names, local addresses, dates in the right format. You install it once and forget about it.

I kept the interface dead simple on purpose. No settings page, no configuration, no tutorial needed. You trigger it, it fills your designs, and you move on.

The Result

Published on the Figma Community. It’s a small tool that fixes a small problem, but it’s a problem every Nepali designer runs into daily. Sometimes the most useful things are the simplest.
